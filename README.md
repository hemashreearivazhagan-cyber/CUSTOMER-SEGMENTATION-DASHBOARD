#  Customer Segmentation Analysis Dashboard

## 📌 Project Overview

This project focuses on segmenting customers based on their demographic and purchasing behavior data. Using Power BI, customers are categorized into different groups to help businesses understand customer patterns and make data-driven marketing decisions.

The dashboard provides interactive visualizations, customer insights, and key performance indicators (KPIs) that support customer analytics and business strategy.

---

## 🎯 Objectives

* Analyze customer demographics and spending behavior.
* Segment customers into meaningful groups.
* Identify high-value, regular, and low-value customers.
* Visualize customer characteristics through interactive dashboards.
* Generate business insights for targeted marketing and customer retention.

---

## 📂 Dataset Information

The dataset contains customer information including:

| Column Name            | Description                      |
| ---------------------- | -------------------------------- |
| CustomerID             | Unique customer identifier       |
| Gender                 | Customer gender                  |
| Age                    | Customer age                     |
| Annual Income (k$)     | Annual income of customer        |
| Spending Score (1-100) | Customer spending behavior score |

---

## 🧹 Data Preprocessing

The following data cleaning steps were performed:

* Checked for missing values
* Verified data formats
* Removed unnecessary columns
* Ensured numerical columns were stored correctly
* Created customer segmentation categories

---

## 👥 Customer Segmentation

Customers were grouped into three categories:

### High Value Customers

* High spending score
* Important target customers
* Potential loyal customers

### Regular Customers

* Moderate spending behavior
* Stable customer base

### Low Value Customers

* Lower spending behavior
* Opportunities for promotional campaigns
